Class: Message

Message(content, sender)

Class representing a message.

Constructor

new Message(content, sender)

Creates a message.
Parameters:
Name Type Description
content String The content of the message
sender Object An object with a name and url property
Example
// Create a message as a bot
import Message from '../js/message.mjs'
const bot = {
   name: 'Your computer',
   url: './images/bot.png'
}
const message = new Message('Hello World!', bot)

Methods

delete()

Deletes the message.
Example
// A say command
import loader from '../js/loader.mjs'
new loader({
  say(message, args) {
    message.reply(args.join(' ')) // Copy the user
    message.delete() // Delete the message!!!!
  }
},{prefix: '~'})

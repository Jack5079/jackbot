Class: Loader

Loader(commands, options)

new Loader(commands, options)

Loads commands and settings.
Parameters:
Name Type Description
commands Object An object of functions.
options Object The options that will be used when loading.
Example

A bot with a say command and the prefix "-"

import Loader from './loader.mjs'
new Loader({
   say (message, args) {
     message.reply(args.join(' '))
   }
 },
 {
   prefix: '-'
})
